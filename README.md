# Projeto de Banco de Dados - Academia FitForce

## Descrição
Este projeto contém os scripts SQL para a criação e manipulação do banco de dados da academia "FitForce". O objetivo é demonstrar a aplicação prática de comandos DML (Insert, Select, Update, Delete) em um cenário real.

## Conteúdo dos Scripts
1. **Inserção de Dados:** Povoamento das tabelas de Planos, Instrutores, Alunos e Aulas.
2. **Consultas:** Relatórios de inadimplência e relação de alunos por plano.
3. **Atualizações:** Ajuste de preços e regularização de pagamentos.
4. **Exclusões:** Remoção de agendamentos cancelados.

## Ferramentas
- SQL (Padrão ANSI/PostgreSQL)
- GitHub para versionamento
